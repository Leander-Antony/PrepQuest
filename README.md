# 📖 Overview

**PrepQuest** is a gamified productivity application designed for developers, students, and aspiring software engineers.

Instead of managing boring task lists, PrepQuest turns your learning journey into an RPG where every coding session, completed project, and interview milestone earns XP.

Build your skills, conquer placement bosses, unlock achievements, and watch your developer journey evolve one quest at a time.

Whether you're solving LeetCode problems, building portfolio projects, preparing for interviews, or learning new technologies, PrepQuest keeps everything organized in one immersive experience.

# 🚀 Getting Started

## Prerequisites

- Node.js 18+
- npm

## Installation

Clone the repository:

```bash
git clone https://github.com/Leander-Antony/prepquest.git
```

Move into the project:

```bash
cd prepquest
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

---

# 🖥 Desktop Version

Run Electron:

```bash
npm run electron:dev
```

Build the Windows executable:

```bash
npm run electron:build
```
